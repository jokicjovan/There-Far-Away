Registration of a new profile

Registration of a user profile can be accessed from the initial view of the application which displays the login to the system. The registration page has the following layout:

  1. Field for entering the user's first name
  2. Field for entering the user's last name
  3. Field for entering the username
  4. Field for entering the password
  5. Registration button
  6. Button for navigating to the login page

To successfully register, users need to fill in all the fields on this page. The username must be unique within the system. The password must be longer than six (6) characters. Clicking the 'Register' button or pressing Enter while a field is in focus will initiate the registration process. Upon successful registration, users are notified and redirected to the login page.

If users wish to sign in to an existing profile instead, they can click the 'Sign in to an existing profile' button, which will take them to the login page.

Shortcut List

  • Move to the next field: Tab
  • Registration (when a field is in focus): Enter
