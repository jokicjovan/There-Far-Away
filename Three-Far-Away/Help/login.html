Login to the System

Login to the system is the first page users encounter when launching the application. The page has the following layout:

  1. Field for entering the username
  2. Field for entering the password
  3. Login button
  4. Button for navigating to the registration page

To successfully login, users need to enter the credentials of an existing user profile into the corresponding fields. After that, clicking the login button or pressing the Enter key will initiate the login process. Upon successful login, users are redirected to a dashboard or a page that provides functionalities specific to their user type. In case users don't have an existing profile, they can click the 'Register now!' button to access the registration page and create a new profile.

Shortcut List

  • Move to the next field: Tab
  • Login (when a field is in focus): Enter
